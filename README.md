# Healthcare Analytics Dashboard

## Overview
This project analyzes healthcare data using Python, Streamlit, and BigQuery.

## Features
- KPI dashboard
- Data visualization (Plotly)
- API integration
- BigQuery support

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- BigQuery

## How to Run
```bash
pip install -r requirements.txt
streamlit run dashboard.py
